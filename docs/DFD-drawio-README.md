# Draw.io / Lucidchart DFD Pack

Files:

- `DFD-drawio.drawio` -> editable multi-page diagram file
- `DFD.md` -> narrative explanation and Mermaid versions

Pages included in `DFD-drawio.drawio`:

- Level 0 - Context
- Level 1 - Main Processes
- Level 2 - Student Self-Service
- Level 2 - Leave Management
- Level 2 - Admin Reporting and Review
- Level 2 - Technical Staff Management
- Level 2 - Warden Operations
- Level 2 - Security Gate Control
- Level 2 - Canteen Operations

How to use in draw.io:

1. Open https://app.diagrams.net/ or the desktop draw.io app.
2. Choose `Open Existing Diagram`.
3. Select `DFD-drawio.drawio`.
4. Edit shapes, colors, arrows, or labels as needed.

How to use in Lucidchart:

1. Open Lucidchart.
2. Import the `DFD-drawio.drawio` file.
3. Lucidchart will convert each page to an editable canvas.

Notes:

- The data stores match the Oracle schema in `database/schema.sql`.
- The flows match the current backend routes and controllers.
- If you need a submission version, export each page as PNG or PDF from draw.io after minor alignment adjustments.
